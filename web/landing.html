SoundCanvas menggabungkan kekuatan cloud storage dengan antarmuka modern untuk pengalaman mendengarkan musik yang sempurna.

☁️

Cloud Storage

Upload file MP3 langsung dari browser. File tersimpan aman di cloud dan bisa diakses kapan saja, di mana saja.

🎵

Smart Playlist

Buat dan kelola playlist sesuai mood Anda. Tambah atau hapus lagu dengan mudah melalui sistem CRUD yang lengkap.

🔒

Secure Authentication

Sistem login yang aman menggunakan Firebase Authentication. Data musik Anda terlindungi dengan enkripsi standar industri.

Operasi CRUD

Full CRUD Operations

Implementasi lengkap Create, Read, Update, Delete pada seluruh entitas menggunakan Firebase Firestore sebagai cloud database.

➕

Create

Upload lagu baru ke cloud beserta metadata (judul, artis, cover). Buat playlist baru dan tambahkan lagu ke dalamnya.

📖

Read

Tampilkan seluruh koleksi musik dari database cloud. Streaming audio langsung dari cloud storage secara real-time.

✏️

Update

Edit metadata lagu (judul, artis, cover image). Update isi playlist — tambah atau kurangi lagu dengan mudah.

🗑️

Delete

Hapus lagu dari library cloud. Hapus playlist beserta seluruh relasinya. Semua operasi tersinkronisasi secara real-time.

🔗

Relasi Many-to-Many

Satu lagu bisa masuk ke banyak playlist. Satu playlist bisa berisi banyak lagu. Diimplementasikan dengan array NoSQL di Firestore.

⚡

Real-time Sync

Semua perubahan langsung tersinkronisasi ke cloud. Data konsisten di semua perangkat yang terhubung.
